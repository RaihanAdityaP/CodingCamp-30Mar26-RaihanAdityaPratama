Expense & Budget Visualizer

A simple web application to track daily expenses and visualize spending using charts. This project is built as part of a 5-day software engineering course.

Overview

Expense & Budget Visualizer is a mobile-friendly web app that helps users:

Track daily transactions
Monitor total balance
Visualize spending distribution by category

The app is designed to be simple, fast, and easy to use without requiring any backend server.

Features (MVP)
1. Input Form
Add new transaction with:
Item Name
Amount
Category (Food, Transport, Fun)
Form validation (all fields required)
2. Transaction List
Displays all transactions
Shows:
Item name
Amount
Category
Delete transaction feature
Scrollable list
3. Total Balance
Displays total balance at the top
Automatically updates when transactions change
4. Visual Chart
Pie chart showing spending distribution by category
Auto updates when data changes
Built using Chart.js (or similar library)
Tech Stack
HTML
CSS
Vanilla JavaScript (no frameworks)
Local Storage (for data persistence)
